# CampusZon Client

Vite + React + TypeScript frontend for CampusZon.

## Setup

Install dependencies:

```bash
cd campuszon-client
npm install
```

Start dev server:

```bash
npm run dev
```

## Built with
- Vite
- React + TypeScript
- Tailwind CSS
- react-router-dom
- axios
- recoil
- socket.io-client
